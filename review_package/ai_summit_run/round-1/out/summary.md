# Round 1 Summary

## 前提
- 実行日: 2026-06-25
- 枠: ② 斬新/未来需要/独創性主体
- ジャンル: AI生活補助
- 重要基準: ヒット可能性、継続収益性、支払い主体の明確さ、Android/iOSネイティブアプリとしての定着力

## 20案一覧
1. ChatGPT / LifeOps Autopilot: 生活の手続き、予約、解約、提出物をAIが次の行動まで分解して進める生活実行エージェント。個人月額と家族共有、将来B2B福利厚生。
2. ChatGPT / Kizuna Keeper: 家族、友人、仕事相手との関係を冷やさないAI関係メンテナンス。人数上限、記念日、贈り物、家族共有で課金。
3. ChatGPT / AI Personal Gatekeeper: 電話、営業DM、怪しいSMS、問い合わせをAIが先に受け止める個人受付。詐欺検知、要約、代理返信で課金。
4. ChatGPT / Care Mesh AI: 子育て、介護、ペット、体調、予定を複数人で引き継ぐAIケア司令塔。家族プラン、対象人数、緊急通知で課金。
5. ChatGPT / Momentum Mate: 片付け、勉強、副業、外出準備などをAIが分解し声かけで完了まで伴走。音声伴走と長時間セッションで課金。
6. Claude / あとで効く: 今の自分から未来の自分へ、AIが文脈ごと音声メッセージを届ける予約アプリ。予約上限、AI再構成、音声演出で課金。
7. Claude / 代わりに悩む: 決断代行ではなく価値観を学習したAIが決めきれない夜に伴走。深掘り対話、決断分析、ナビゲーターで課金。
8. Claude / わたしの通訳: 相手、関係性、温度に合わせて送信前メッセージを翻訳。相手登録、謝罪/交渉など高難度モードで課金。
9. Claude / ひとり会議: 脳内の複数の自分を別人格AIに分離し会議させる内省アプリ。人格数、カスタム、議事録、音声化で課金。
10. Claude / 未来日記: 予定、気分、行動パターンからAIが明日の自分を先回りして日記化。予測、振り返り、人格化で課金。
11. Gemini / Gimme: 補助金、控除、解約忘れなど個人のもらい損ねを検知して奪還するエージェント。月額監視または成果報酬。
12. Gemini / NoMan: 断る、退会する、交渉するを角を立てずに代行する悪役AI。テキスト生成、電話退会、交渉チケットで課金。
13. Gemini / FamiLogue: 自分のAIと親のAIが裏で対話し実家とのすり合わせを行う外交エージェント。親子サブスクとギフト送客。
14. Gemini / StockDrop: スクショやURLのあとで見る情報を予定、買い物、予約など物理行動へ変換。月額と送客手数料。
15. Gemini / OneTrack: マルチタスク麻痺を1つの指示だけに直列化する視覚封鎖型タスクナビ。サブスクと個人適応エンジン。
16. Grok / フューチャー・レター・エージェント: 現在の自分と未来の自分が手紙や返信で対話する長期視点アプリ。音声会話、未来シナリオ、履歴分析で課金。
17. Grok / リレーション・シンクロナイザー: 人間関係をAIがマッピングし関わりタイミングとメッセージを提案。関係健康スコア、複数人管理、家族共有で課金。
18. Grok / デイ・リハーサル・スタジオ: 明日の会話や重要イベントをAIで事前シミュレーション。無制限リハーサル、感情フィードバック、音声練習で課金。
19. Grok / ライフ・レイヤー・インサイト: 人生領域をレイヤー化し相互影響と全体最適アクションを提案。詳細分析、週次/月次レポート、目標追跡で課金。
20. Grok / エンパシー・ミラー: 大切な人の心のモデルを作り、自分の言動を相手視点でシミュレーション。個人/カップル/家族共有で課金。

## Provider別傾向
- ChatGPT: 生活実務、関係管理、ケア共有など、日常インフラ化しやすい案が多い。支払い主体は比較的明確だが、既存カテゴリに寄る案もある。
- Claude: 未来の自分、決断、関係性の言語化、内省など感情的フックが強い。バズ性はあるが、継続課金の支払い理由をより具体化したい。
- Gemini: もらい損ね奪還、退会/交渉代行、スクショ行動化、タスク直列化など、痛みと支払い理由が鋭い。リスクは大きいが伸びしろも大きい。
- Grok: 自己対話、関係性、未来予行、人生統合など広めの自己改善軸。体験設計次第では強いが、汎用AI内省ツール化に注意。

## 次ラウンド用整理メモ
- ヒット可能性が高い案: Gimme、StockDrop、OneTrack、AI Personal Gatekeeper、わたしの通訳、NoMan。
- 継続収益性が見える案: Gimme、Care Mesh AI、OneTrack、Kizuna Keeper、FamiLogue、エンパシー・ミラー。
- 支払い主体が明確な案: Gimme、NoMan、AI Personal Gatekeeper、Care Mesh AI、OneTrack、わたしの通訳。
- 需要が強い案: 個人の面倒ごと代行、詐欺/営業防御、家族ケア、関係性摩擦、マルチタスク麻痺、補助金/解約忘れ。
- 独自性がある案: StockDrop、OneTrack、あとで効く、FamiLogue、NoMan、ひとり会議。
- リスクはあるが伸びしろが大きい案: Gimme、NoMan、StockDrop、FamiLogue、エンパシー・ミラー、AI Personal Gatekeeper。
- 汎用的すぎて危ない案: LifeOps Autopilot、ライフ・レイヤー・インサイト、フューチャー・レター・エージェント、未来日記。
