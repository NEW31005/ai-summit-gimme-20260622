## 今回の主な修正

- 児童手当をこども家庭庁ルール基準へ寄せた
  - 18〜22歳で監護相当・生計費負担がある上の子を入力できる
  - 第3子以降判定に上の子を含める
  - 第1子・第2子は3歳未満かどうか未入力のため月1万円〜1万5千円レンジ
  - 第3子以降は月3万円で年額化
- 医療費控除を「医療費がそのまま戻る」表現から分離
  - 医療費控除額の式をモデル化
  - 所得税・住民税軽減の目安として幅を持たせて表示
- AI明細スキャンを強化
  - 有名サービス名だけに依存せず、金額行を広く候補化
  - 月額・年額・週額・四半期を検出し月換算
  - 周期未確定、元金額、月換算、確信度、理由を表示
- Plusの継続収益価値をモデルに追加
  - Freeは上位2件に制限
  - Plusは全候補、詳細根拠、毎月の取りこぼし監視、通知、家族レポート、証跡保存の価値に寄せた
- テスト更新
  - 18〜22歳の上の子を含む第3子判定
  - 年払いサブスクの月換算
  - 医療費控除額と税軽減見込みの分離
