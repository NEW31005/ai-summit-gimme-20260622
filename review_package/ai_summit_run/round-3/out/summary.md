# Round 3 Summary

## 採用する強化要素

### 1. Gimme / Gemini
- 採用強化: 世帯奪還マトリクス + 損失ウィジェット + 奪還クエスト
- 核: 個人ではなく家族/世帯単位で補助金、控除、解約忘れ、申請期限を横断監視し、最適な申請ルートを提示する。
- 収益強化: 家族まるごと監視プラン、成果報酬、奪還金額の可視化で高単価・低解約化。
- 注意リスク: 所得/家族情報の同意管理、公的制度の追従、申請代行の責任範囲。

### 2. NoMan / Gemini
- 採用強化: 1タップ撃退チケット + 法的テンプレ付き着信/通話転送 + NoMan成長システム
- 核: ユーザーが嫌な勧誘、退会引き止め、交渉を1タップでAIへ渡し、法的・心理的にブレない断りを実行する。
- 収益強化: 月額基本 + 緊急撃退チケット + 電話代行パック + キャラ/強さカスタム。
- 注意リスク: 過剰な表現、相手を刺激する文面、通話録音/転送、悪用防止。

### 3. StockDrop / Gemini
- 採用強化: 今日の魔法一タップ変換 + ゴースト・コンシェルジュ仮押さえ
- 核: スクショ/URL/あとで見るをAIが予定、買い物、予約、移動、タスクへ変換し、必要なら短時間だけ仮押さえする。
- 収益強化: 月額 + 予約/購入/地図/ネットスーパー送客手数料 + 優先変換プレミアム。
- 注意リスク: プライバシー、外部サービス規約、誤予約/誤購入、広告臭。

### 4. OneTrack / Gemini
- 採用強化: 今これだけモード + ディープ・フォーカス・カプセル + 脳特性適応コーチ
- 核: AIが全タスクを預かり、画面/通知/音声/ウィジェットを「今やる1個」だけに封鎖する。
- 収益強化: 月額サブスク、個人適応エンジン、集中レポート、音声伴走、集中パック。
- 注意リスク: 強制感、他アプリ遮断のOS制限、メンタルヘルス表現、緊急通知の見逃し。

### 5. AI Personal Gatekeeper / ChatGPT
- 採用強化: 親スマホ防衛プラン + 家族安心ガード + 今日の防御スコア
- 核: 本人だけでなく高齢親の電話、SMS、DM、詐欺っぽい連絡をAIが一次防御し、子側へ要約通知する。
- 収益強化: 家族見守り月額、親追加、危険連絡アーカイブ、詐欺パターン更新、安心スコア。
- 注意リスク: 通話/SMS連携、本人同意、誤ブロック、重要連絡の見落とし、プライバシー。

## Round 4への統合メモ
- Gimmeは金銭的便益が最も直感的で、全LLMが本命視。Champion候補の中心。
- NoManは日本的な「断れない」痛みに刺さる。キャラ性とチケット課金で尖れる。
- StockDropはAI時代の情報過多を行動に変える新しさが強い。送客収益で伸びる。
- OneTrackは日常定着が最も強くなりうるが、アプリ制作では体験設計がすべて。
- AI Personal Gatekeeperは「親を守る」不安課金が強い。家族向け防御インフラに寄せるほど収益性が上がる。
